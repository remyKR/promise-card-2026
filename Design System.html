Primitives — Neutral

Cool-gray scale used for text, borders, placeholders, and panel surfaces. Pulled directly from the editor's input labels and dividers.

Primitives — Gold (brand accent)

The brand's signature hue — used on theme pagination, serif wordmarks, dividers, and hover states. Derived from the Figma source's logo ink (#7B4401) and card lettering (#CDA66E).

Primitives — Cream / Ivory

Warm surface tones that define the editor canvas and header band.

Primitives — Theme Swatches

The 7 pastel dots users pick from to tint their invitation. Kept as primitives so they can be cross-referenced by theme presets.

Token JSON (W3C-style)

design-tokens.community-group

Full JSON export is available in tokens.css. Snippet below shows the structure — each token carries $value and $type for Figma Variable import. 

{
  "color": {
    "gold": {
      "500": { "$value": "#CDA66E", "$type": "color" },
      "700": { "$value": "#7B4401", "$type": "color" }
    },
    "text": {
      "primary":     { "$value": "{color.gray.900}", "$type": "color" },
      "placeholder": { "$value": "{color.gray.500}", "$type": "color" }
    }
  },
  "font": {
    "family": {
      "display": { "$value": "SVN-Poppins", "$type": "fontFamily" }
    },
    "size": {
      "base": { "$value": "14", "$type": "dimension" },
      "2xl":  { "$value": "26", "$type": "dimension" }
    }
  },
  // ... spacing, radius, shadow follow same pattern
}
