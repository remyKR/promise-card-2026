# 서비스 개요

> 버전: v2.0 | 최종수정: 2026-04-23

---

## 1. 서비스 목적

베트남에서 결혼을 준비하는 커플이 **5분 안에 모바일 청첩장을 만들고 공유**할 수 있는 서비스.
종이 청첩장의 비용·배송 문제를 해결하고, Zalo·Facebook 등 현지 메신저로 즉시 공유 가능.

---

## 2. 타겟 사용자

| 구분 | 내용 |
|---|---|
| 1순위 | 베트남인 커플 (베트남인 × 베트남인) |
| 2순위 | 베트남 거주 한국인 커플 |
| 비대상 | 한국인 × 베트남인 혼합 커플 |
| 페르소나 | 20~30대 초반, 호치민·하노이 거주, 스마트폰 중심 생활 |
| 핵심 불편 | 종이 청첩장 비용, 신부측·신랑측 2회 예식에 각각 다른 청첩장 필요 |

---

## 3. 핵심 가치

| 가치 | 설명 |
|---|---|
| 간편함 | 5분 내 완성, 복잡한 디자인 도구 불필요 |
| 현지화 | 베트남어 UI, 베트남 지도·메신저 연동 |
| 모바일 최적화 | Zalo·Facebook으로 링크 한 번에 공유 |
| 비용 절감 | MVP 무료, 종이 청첩장 대비 0원 |

---

## 4. 성공 지표 (MVP 기준)

| 지표 | 목표 |
|---|---|
| 청첩장 생성 수 | 월 100건 |
| 공유 클릭률 | 80% 이상 |
| 재방문율 | 30% 이상 |
| 서비스 오류율 | 1% 미만 |

---

| 리스크 | 대응 |
