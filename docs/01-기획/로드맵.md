## MVP 목표

6주 안에 서비스 출시. 기획·정책·설계 완료 후 구현 시작.

---

## 주차별 계획

| 주차 | 기간 | 목표 | 핵심 산출물 |
|---|---|---|---|
| **1주** | 환경 세팅 | Supabase 연결, Google·Facebook·OTP 로그인 | 로그인 화면, 전화번호 인증 |
| **2주** | 에디터 기본 | 청첩장 에디터 폼 (기본정보·일시·장소) | 에디터 화면 (P1 섹션) |
| **3주** | 템플릿·미리보기 | 커버 템플릿 3종, 실시간 미리보기 | 템플릿 렌더러 |
| **4주** | 갤러리·지도·공유 | 사진 업로드 20장, Google Maps, 공유 URL | 갤러리, 지도 컴포넌트 |
| **5주** | RSVP·방명록·마이페이지 | 참석 여부 수집, 방명록, 내 청첩장 목록 | RSVP·방명록 폼, 마이페이지 |
| **6주** | 배포·버그수정 | Vercel 배포, QA, 오류율 1% 미만 | 프로덕션 URL |
