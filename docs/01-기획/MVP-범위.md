## 종이 청첩장 통합 (v2~v3, 3단계 도입)

> 상세 명세: [03-설계/화면명세/07-종이청첩장.md](../03-설계/화면명세/07-종이청첩장.md)

### 도입 배경
- 베트남 도시 커플 **40~60%** 종이 청첩장 사용 (어른·친척 전달 문화)
- 베트남 5개 경쟁사 중 **모바일+종이 통합 사례 없음** → 블루오션
- **매출 7~10배** 증가 (디지털 200K~500K vs 패키지 1.5M~3.5M VND)
- **모방 방어막**: 인쇄소 파트너십·물류 노하우 6~12개월 진입 장벽

### Phase 2 (출시 6~9개월): DIY PDF 다운로드

| 기능 | 등급 | 비고 |
|---|---|---|
| 자동 사이즈 변환 (A6/A5/바이폴드/트라이폴드) | v2 | 모바일 → 종이 레이아웃 |
| 종이용 디자인 자동 변환 | v2 | CMYK·여백·재단선 자동 |
| 표지·속지 콘텐츠 항목 선택 | v2 | 사용자 자유 선택 |
| 미리보기 (2D 평면) | v2 | |
| 인쇄용 PDF 다운로드 | v2 | CMYK 300 DPI Bleed 3mm, 사용자가 직접 인쇄소 발주 |

### Phase 3 (출시 12~18개월): 인쇄소 파트너십 + 표준 옵션

| 기능 | 등급 | 비고 |
|---|---|---|
| 종이 재질 선택 (매트/글로스/펄/특수지) | v3 | |
| 수량 선택 (50/100/200/500장+) | v3 | 단가 차등 |
| 봉투 디자인·인쇄 | v3 | 게스트 이름 mail merge 옵션 |
| 결제 (ZaloPay/MoMo/VietQR/카드) | v3 | 결제 PG 도입과 동시 |
| 인쇄소 자동 발주 | v3 | 호치민·하노이 파트너 1곳씩 |
| 본인쇄 + 베트남 전국 배송 | v3 | GHN/Viettel Post/J&T |
| 배송 추적 (Zalo/SMS 알림) | v3 | |

### 페이지 수별 유형 3종 (기능 고정)

사용자가 페이지 수 먼저 선택 → 해당 유형의 고정 기능만 노출.
상세: [03-설계/화면명세/07-종이청첩장.md](../03-설계/화면명세/07-종이청첩장.md) 섹션 1-A

| 유형 | 페이지 | 사이즈 | 등급 | 핵심 기능 |
|---|---|---|---|---|
| **①** | 2페이지 | A6 / A5 (1장 양면) | 미니멀 | 이름·사진·날짜 + 메인 청첩장 QR + VietQR(선택) |
| **②** ⭐ | 4페이지 | A5 2장 / 바이폴드 | 표준 | + 인사말 + 일시·장소 상세 + 메인 QR + VietQR + RSVP QR(선택) |
| **③** | 6+페이지 | A5 3장+ / 트라이폴드 / 소책자 | 프리미엄 | 모든 기능 + 갤러리 + QR 7종 |

### 모바일 → 종이 매핑 (3가지 카테고리)

상세: [07-종이청첩장.md](../03-설계/화면명세/07-종이청첩장.md) 섹션 3

1. **직접 변환 (자동 채움)** — 모바일 데이터 → 종이 자동 (이름·사진·일시·장소·인사말·갤러리 등)
2. **QR 코드 통합** ⭐ — 종이 정적 한계를 모바일로 보완 (7종 QR: 메인·RSVP·방명록·갤러리·음악·지도·VietQR)
3. **종이 미적용** — 정적 매체 한계 (배경 효과·D-Day·갤러리 타입·공유 카드)
