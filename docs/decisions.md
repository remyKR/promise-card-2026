### 디자인 시스템 토큰 확장
- **결정**: 다음 토큰 추가
  - `number/56` (Primitives - Numbers)
  - `size/xl` 매핑 변경 (`number/64` → `number/56`)
  - `spacing/56` (Semantic, alias to `number/56`)
  - `heading/13` 텍스트 스타일 (font-size 13, SVN-Poppins SemiBold)
- **이유**: 기존 size 스케일에 56 단계 부재. 컴포넌트 디자인 진행 중 필요성 확인.
- **참고**: 기존 `size/xl=64`는 디자인에서 거의 미사용 → 56으로 재매핑하여 t-shirt 스케일 자연화 (32/40/44/56/80)

### Zalo 알림 P3(v2) → P2(MVP) 격상
- **결정**: RSVP 제출 시 신랑·신부에게 Zalo 메시지 알림 발송 기능을 MVP P2로 격상
- **이유**: Zalo는 베트남 메신저 시장 점유율 ~80%로 카카오톡 같은 표준 인프라. RSVP 응답 추적의 핵심 기능. 누락 시 베트남 사용자 경험 크게 떨어짐.
- **구현 단계**:
  - **MVP (P2)**: RSVP 제출 즉시 알림, 알림 빈도 설정 (즉시/일일/끄기), Zalo 번호 OTP 인증
