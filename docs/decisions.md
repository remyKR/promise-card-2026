### 디자인 시스템 토큰 확장
- **결정**: 다음 토큰 추가
  - `number/56` (Primitives - Numbers)
  - `size/xl` 매핑 변경 (`number/64` → `number/56`)
  - `spacing/56` (Semantic, alias to `number/56`)
  - `heading/13` 텍스트 스타일 (font-size 13, SVN-Poppins SemiBold)
- **이유**: 기존 size 스케일에 56 단계 부재. 컴포넌트 디자인 진행 중 필요성 확인.
- **참고**: 기존 `size/xl=64`는 디자인에서 거의 미사용 → 56으로 재매핑하여 t-shirt 스케일 자연화 (32/40/44/56/80)

### Zalo 알림 P3(v2) → P2(MVP) 격상
- **결정**: RSVP 제출 시 신랑·신부에게 Zalo 메시지 알림 발송 기능을 MVP P2로 격상
- **이유**: Zalo는 베트남 메신저 시장 점유율 ~80%로 카카오톡 같은 표준 인프라. RSVP 응답 추적의 핵심 기능. 누락 시 베트남 사용자 경험 크게 떨어짐.
- **구현 단계**:
  - **MVP (P2)**: RSVP 제출 즉시 알림, 알림 빈도 설정 (즉시/일일/끄기), Zalo 번호 OTP 인증

  - 옵셋급 품질 가능 (해상도 812 DPI·메탈릭/화이트 잉크)
  - mail merge (게스트별 답례품 라벨·봉투 이름 인쇄) = 디지털 필수
  - 50~500장 소량 가격 효율적 (옵셋은 셋업비로 비효율)
  - 빠른 납기 (1~3일)
- **5단계 가격 옵션 (50장 기준)**:
  - 베이직: 디지털 + 코팅 (1.5M~2M VND)
  - 메탈릭: + 메탈릭 잉크 (2M~2.5M VND)
  - 박찍기: + 박찍기 (2.5M~3.5M VND)
  - 풀 럭셔리: + 박+엠보싱+다이컷 (4M~6M VND)
  - 초프리미엄: + 활판 인쇄 (6M~10M VND)
- **변경 파일**: [02-정책/서비스-정책.md 섹션 8](02-정책/서비스-정책.md), [07-종이청첩장.md](03-설계/화면명세/07-종이청첩장.md)

### 후가공 1장 샘플 — 3 Tier 시스템
- **결정**: Phase 4 프리미엄 옵션의 테스트 인쇄를 3 Tier로 분리
  - **Tier 1 (디지털 샘플)**: 디지털 1장 + 후가공 디지털 시뮬레이션 / +200K~500K VND / 3~5일
  - **Tier 2 (디지털 + 표준 후가공 키트)**: 디지털 1장 + 인쇄소 보유 박·엠보싱 표준 샘플 / +400K~800K VND / 5~7일
  - **Tier 3 (풀 샘플)**: Die 제작 + 후가공 통합 1장 실물 / +1M~3M VND / 10~14일 (본 인쇄 시 Die 재사용 → 본 단가 차감)
- **이유**: 후가공 Die 제작비 때문에 1장 샘플 단가가 본 인쇄 50장과 비슷. 사용자 주문 규모·필요도에 맞춘 단계화 필요.
- **권장 매트릭스**:
  - 50~300장: Tier 1
  - 300~500장 + 박찍기·엠보싱: Tier 2
  - 1,000장+ 또는 풀 후가공: Tier 3

### 프리미엄 템플릿 시스템 — 프리프린트 + 데이터 후인쇄 채택 ⭐
- **결정**: 베트남 청첩장 시장 표준 모델 채택 — 인쇄소가 박·형압된 종이를 미리 대량 가공·재고 보관 → 사용자 주문 시 데이터만 디지털 후인쇄
- **이유**:
  - 베트남 청첩장 시장 70~80%의 표준 모델 (사용자 친숙)
  - Die 비용 분산 → 사용자 단가 30~50% 절감
  - 소량(50장)에도 박·형압 가능 → 진입 장벽 ↓
  - 빠른 납기 (1~3일)
  - 매출 잠재력 (Phase 3 안정 시 월 530M VND ≈ 2,900만원)
- **3 Tier 가격**:
  - 베이직 (디지털만): 1.5~2M VND
  - **프리미엄 템플릿 (프리프린트)** ⭐: 2~3M VND (사용자 70% 예상)
  - 풀 커스텀 (전용 Die): 4~6M VND
- **Phase별 라인업**: Phase 3 (8~10종) → Phase 4 (20~30종) → Phase 5 (50종+)
- **초기 투자**: Die + 종이 사전 가공 약 50~100M VND (우리 부담분)
- **변경 파일**: [07-종이청첩장.md](03-설계/화면명세/07-종이청첩장.md)

### 인쇄소 협력 모델 — 옵션 A 채택 (인쇄 전문, 우리 디자인) ⭐
- **결정**: 인쇄소에 **인쇄·후가공·배송만** 의뢰. 디자인은 Promise Card가 책임.
- **두 가지 옵션 비교**:
  - 옵션 A (인쇄 전문 업체): 우리 디자인 통제 100% / 모바일↔종이 일관성 ✅ / 마진 ↑ / 우리 브랜드 자산
  - 옵션 B (디자인+인쇄 통합 업체): 디자인 비용 X / 마진 ↓ / 우리 브랜드 약화 / 모바일↔종이 일관성 ❌
- **이유**:
  - 우리 핵심 가치 = "모바일 청첩장 → 종이 청첩장 자동 변환" → **디자인이 핵심 자산**
  - 인쇄 통합 업체 사용 시 모바일·종이 디자인 일관성 깨짐
  - 마진 확보 (디자인 가치는 우리가 가져감)
  - 여러 인쇄소 활용 가능 (한 업체 종속 X)
- **단계적 전략**:
  - Phase 2~3: 옵션 A 100% (투득 인쇄 공장 우선)
  - Phase 4 럭셔리 라인: 부티크 스튜디오와 한정 콜라보 (예: "Promise x Sweet Dates 컬렉션")
  - Phase 5+: 자체 인쇄 인프라 또는 인쇄 전문 업체 인수 검토
