# 청첩장 제작하기 — 화면 기능 명세서

> 경로: `/create`  
> 인증: 로그인 필요  
> 레퍼런스: 데어무드 모바일청첩장 제작하기  
> 버전: v1.0 | 작성일: 2026-04-23

---

## 화면 구조

```
┌─────────────────────────────────────────┐
│  [미리보기 영역]   │  [설정 패널]         │
│  (좌측 or 상단)   │  (우측 or 하단)      │
│                   │  섹션 탭 목록        │
│  실시간 렌더링     │  각 섹션 설정 폼     │
└─────────────────────────────────────────┘
```

- **모바일**: 상단 미리보기(50%) + 하단 스크롤 설정 패널
- **데스크톱**: 좌측 미리보기(고정) + 우측 설정 패널(스크롤)
- 설정값 변경 시 미리보기 **실시간 반영**

## 섹션 1. 테마·디자인

### 1-1. 글꼴 선택
- **작동**: 글꼴 목록에서 하나를 탭하면 청첩장 전체 텍스트에 즉시 적용됨
- **옵션**: 고운돋움 / 리디바탕 / 이롭게바탕 / 둥근바람 / 연성체 / 노토산스
- **크기**: 보통 / 크게 / 더 크게 — 기본값은 "보통"
- **미리보기**: 글꼴 목록 각 항목에 해당 폰트로 렌더링된 샘플 텍스트 표시
- **MVP**: ✅ 포함 | **우선순위**: P1

### 1-2. 배경 색상
- **작동**: 색상 피커(또는 프리셋 팔레트)에서 색상 선택 → 청첩장 전체 배경색 변경
- **기본값**: 흰색(#FFFFFF)
- **제한**: 가독성 보장을 위해 너무 어두운 색 선택 시 텍스트 색상 자동 반전 고려
- **MVP**: ✅ 포함 | **우선순위**: P1

### 1-3. 강조 색상
- **작동**: 소제목, 구분선, 버튼 등 포인트 요소에 적용되는 색상 선택
- **연동**: 배경 색상과 자동 대비 계산하여 가독성 유지
- **MVP**: ✅ 포함 | **우선순위**: P2

### 1-4. 배경 패턴
- **작동**: 청첩장 배경 전체에 반복 패턴 이미지를 오버레이로 적용
- **옵션**: 없음 / 작은 꽃 / 큰 꽃
- **투명도**: 패턴은 반투명(약 10~20% opacity)으로 텍스트 가독성 유지
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 1-5. 배경 이펙트
- **작동**: 청첩장 화면 위에 떨어지는 파티클 애니메이션 지속 재생
- **옵션**: 없음 / 벚꽃 / 눈 / 낙엽 / 개나리 / 안개꽃
- **성능**: CSS 애니메이션 또는 Canvas 기반 — 저사양 기기 고려 필요
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 1-6. 등장 효과
- **작동**: 게스트가 청첩장을 스크롤할 때 각 섹션이 부드럽게 등장하는 애니메이션
- **옵션**: 없음 / 샤르륵(fade-in + slide-up)
- **구현**: Intersection Observer API 활용
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 1-7. 확대 방지
- **작동**: 게스트 화면에서 핀치 줌 및 더블탭 확대를 비활성화
- **구현**: `<meta name="viewport" content="user-scalable=no">` 동적 삽입
- **목적**: 청첩장 레이아웃 깨짐 방지, 사진 저작권 보호
- **MVP**: ✅ 포함 | **우선순위**: P2

### 1-8. 청첩장 잠금
- **작동**: 공개 URL 접근 제어 설정
- **옵션**:
  - 공개: 누구나 접근 가능 (기본값)
  - 예식 후 잠금: 설정한 예식일 +1일 00:00부터 자동으로 비공개 전환
  - 바로 잠금: 즉시 비공개 (링크 접근 시 "청첩장이 종료되었습니다" 안내)
- **MVP**: ✅ 포함 | **우선순위**: P2

## 섹션 2. 기본 정보

### 2-1. 신랑·신부 이름 입력
- **작동**: 각각 성(họ)과 이름(tên) 분리 입력 또는 전체 이름 입력
- **표시**: 청첩장 메인 화면과 상단 헤더에 "신랑이름 & 신부이름" 형태로 표시
- **유효성 검사**: 빈값 저장 방지, 최대 20자 제한
- **MVP**: ✅ 포함 | **우선순위**: P1

### 2-2. 부모님 이름 입력
- **작동**: 신랑/신부 각각 아버지·어머니 이름 입력 (선택 사항)
- **표시**: 청첩장 하단 혼주 안내 영역에 "신랑 아버지 ○○○ · 어머니 ○○○" 형태
- **생략 처리**: 미입력 시 해당 라인 자동 숨김
- **MVP**: ✅ 포함 | **우선순위**: P1

### 2-3. 故人 표기
- **작동**: 故人 체크박스 선택 시 해당 이름 앞에 "故" 텍스트 또는 국화 아이콘 표시
- **옵션**: 텍스트(故) 표기 / 국화 이미지 표기
- **MVP**: ✅ 포함 | **우선순위**: P2

### 2-4. 세례명 입력
- **작동**: 신랑·신부·부모님 각각 세례명 별도 입력 가능
- **표시**: 이름 옆에 소괄호로 "이름 (세례명)" 형태 표시
- **생략 처리**: 미입력 시 자동 숨김
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 2-5. 신부측 먼저 표시
- **작동**: 토글 ON 시 청첩장 전체에서 신부 정보가 신랑 정보보다 앞에 표시
- **적용 범위**: 혼주 안내, 이름 표기, 계좌번호 순서 등 전체 연동
- **MVP**: ✅ 포함 | **우선순위**: P2

## 섹션 3. 메인 화면

### 3-1. 커버 디자인 선택
- **작동**: 청첩장 최상단 커버 영역의 레이아웃 선택
- **옵션**:
  - 기본: 사진 위에 텍스트 오버레이
  - 채우기: 사진이 화면 전체를 꽉 채움
  - 아치: 상단 아치형 프레임 안에 사진 배치
  - 타원: 타원형 프레임 안에 사진 배치
  - 액자: 사각 테두리 액자 스타일
- **MVP**: ✅ 3종 이상 포함 | **우선순위**: P1

### 3-2. 커버 사진 이펙트
- **작동**: 커버 사진에 시각적 필터 효과 적용
- **옵션**: 없음 / 안개(흰색 그라디언트 오버레이) / 물결(blur) / 페이퍼(종이 질감)
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 3-3. 인사말 문구
- **작동**: 청첩장 본문 상단에 표시되는 초대 문구 입력
- **기본 샘플**: 미리 작성된 샘플 문구를 버튼 클릭으로 자동 입력
- **직접 입력**: 제목(선택) + 본문(필수) 개별 입력
- **글자 수 제한**: 본문 최대 500자
- **MVP**: ✅ 포함 | **우선순위**: P1

### 3-4. 성함 표기 위치
- **작동**: 커버 이미지와 인사말 사이에 신랑·신부 이름 표시 위치 선택
- **옵션**: 사진 위 / 인사말 아래
- **MVP**: ✅ 포함 | **우선순위**: P2

## 섹션 4. 예식 일시

### 4-1. 날짜 선택
- **작동**: 달력 UI에서 날짜 탭 → 선택된 날짜가 청첩장에 자동 포맷으로 표시
- **표시 포맷**: "2026년 4월 22일 수요일" (자동 요일 계산)
- **최솟값**: 오늘 이후 날짜만 선택 가능
- **MVP**: ✅ 포함 | **우선순위**: P1

### 4-2. 시간 선택
- **작동**: 시(오전7시~오후11시) + 분(00~55분, 5분 단위) 드롭다운으로 선택
- **표시 포맷**: "오후 6시 30분" (베트남어: "18 giờ 30 phút")
- **MVP**: ✅ 포함 | **우선순위**: P1

### 4-3. 표시 방식 — 캘린더
- **작동**: 청첩장 내 한 달 달력이 표시되며 예식일에 표시
- **스타일**: 미니 달력, 예식일 강조 표시(원형 배경)
- **MVP**: ✅ 포함 | **우선순위**: P2

### 4-4. 표시 방식 — D-Day 카운트다운
- **작동**: 예식일까지 남은 일수·시간·분·초를 실시간으로 카운트다운
- **포맷**: "DAYS 00 : HOURS 00 : MIN 00 : SEC 00"
- **예식 당일**: "○○의 결혼식이 오늘입니다!" 문구로 전환
- **예식 후**: 카운트다운 영역 숨김 또는 "결혼을 축하합니다" 문구 표시
- **MVP**: ✅ 포함 | **우선순위**: P2

### 4-5. D-Day 문구 커스텀
- **작동**: "○○와 ○○의 결혼식이 D-Day 일 남았습니다." 문구에서 앞부분 직접 입력
- **변수**: `{신랑이름}`, `{신부이름}`, `{D-Day}` 자동 치환
- **MVP**: ✅ 포함 | **우선순위**: P2

## 섹션 5. 예식 장소

### 5-1. 주소 검색
- **작동**: 검색창에 예식장명 또는 주소 입력 → Google Maps API로 자동완성 목록 표시 → 선택 시 위도·경도·주소 자동 입력
- **국내/해외 전환**: 베트남 기준 국내(VN) 우선, 해외 직접 입력도 지원
- **MVP**: ✅ 포함 | **우선순위**: P1

### 5-2. 예식장 상세 정보 입력
- **항목**: 예식장명 / 층과 홀 이름 / 대표 연락처
- **표시**: 청첩장 오시는 길 섹션 상단에 텍스트로 표시
- **MVP**: ✅ 포함 | **우선순위**: P1

### 5-3. 지도 표시
- **작동**: 토글 ON 시 청첩장 내 Google Maps 지도 삽입
- **기능**: 핀 마커 표시, 확대/축소 조작 (지도 잠금 시 스크롤만 가능)
- **지도 높이**: 기본(250px) / 축소(150px), 줌 레벨 15~19 선택
- **MVP**: ✅ 포함 | **우선순위**: P1

### 5-4. 지도 잠금
- **작동**: 토글 ON 시 지도 드래그·핀치·클릭 비활성화
- **목적**: 게스트가 실수로 지도를 조작해 페이지 스크롤이 방해받는 것 방지
- **MVP**: ✅ 포함 | **우선순위**: P2

### 5-5. 내비게이션 앱 바로가기
- **작동**: 버튼 탭 시 기기에 설치된 지도 앱(Google Maps / Apple Maps / Zalo Maps)으로 예식장 좌표 전달하여 길 안내 시작
- **베트남**: Google Maps 딥링크 우선
- **MVP**: ✅ 포함 | **우선순위**: P1

### 5-6. 교통 수단 안내
- **작동**: 최대 4개 교통 수단 정보를 자유 텍스트로 입력
- **표시**: 아코디언 또는 탭 형태로 게스트 화면에 표시
- **예시**: "버스 — 105번, 205번 탑승 후 ○○역 하차"
- **MVP**: ✅ 포함 | **우선순위**: P2

### 5-7. 약도 사진
- **작동**: 직접 제작한 약도 이미지 파일 업로드 → 지도 아래에 표시
- **지원 형식**: JPG, PNG, WebP
- **MVP**: ✅ 포함 | **우선순위**: P2

## 섹션 6. 갤러리

### 6-1. 사진 업로드
- **작동**: 파일 선택 또는 드래그&드롭으로 최대 40장 업로드
- **지원 형식**: JPG, PNG, WebP, HEIC
- **자동 최적화**: 업로드 시 Supabase Storage에 저장, WebP 변환 권장
- **순서 변경**: 업로드 후 썸네일 드래그로 순서 재정렬
- **MVP**: ✅ 포함 (최대 20장으로 제한) | **우선순위**: P1

### 6-2. 갤러리 타입
- **작동**: 게스트 화면에서 사진을 표시하는 레이아웃 방식 선택
- **옵션**:
  - 스와이프: 한 장씩 좌우 슬라이드
  - 썸네일: 3열 그리드 썸네일 → 탭 시 크게 보기
  - 그리드: 2열 그리드 전체 표시
- **MVP**: ✅ 스와이프 + 그리드 포함 | **우선순위**: P1

### 6-3. 팝업 뷰어
- **작동**: 사진 탭 시 전체 화면 팝업으로 원본에 가까운 크기로 표시
- **제스처**: 좌우 스와이프로 이전/다음 사진 이동, 위아래 스와이프로 닫기
- **MVP**: ✅ 포함 | **우선순위**: P2

### 6-4. 줌 기능
- **작동**: 팝업 뷰어 내에서 핀치 투 줌으로 사진 확대 가능 (토글로 ON/OFF)
- **MVP**: ✅ 포함 | **우선순위**: P2

### 6-5. 엔딩 사진·문구
- **작동**: 갤러리 마지막에 대표 사진 1장 + 감사 문구를 엔딩 카드로 표시
- **문구**: 기본 샘플 또는 직접 입력
- **MVP**: ✅ 포함 | **우선순위**: P2

## 섹션 7. 계좌번호

### 7-1. 계좌 그룹 관리
- **작동**: 신랑측 / 신부측 그룹으로 나뉘며 각 그룹에 계좌 여러 개 추가 가능
- **그룹 추가**: "그룹 추가" 버튼으로 부모님 계좌 등 별도 그룹 생성
- **MVP**: ✅ 포함 | **우선순위**: P1

### 7-2. 계좌 정보 입력
- **항목**: 예금주명 / 은행명 / 계좌번호
- **자동 포맷**: 계좌번호 입력 시 은행별 하이픈 자동 삽입
- **MVP**: ✅ 포함 | **우선순위**: P1

### 7-3. 노출 타입
- **작동**: 게스트 화면에서 계좌 정보를 보여주는 UI 방식 선택
- **옵션**:
  - 아코디언: 그룹명 탭 시 계좌 목록 펼쳐짐/접힘
  - 스와이프: 좌우 슬라이드로 그룹 전환
  - 카드: 모든 계좌 카드 형태로 펼쳐서 표시
- **MVP**: ✅ 아코디언 포함 | **우선순위**: P2

### 7-4. 간편송금 연동
- **작동**: 계좌번호 옆에 간편송금 버튼 표시 → 탭 시 해당 앱으로 이동
- **베트남**: MoMo / ZaloPay / VietQR 지원 (MVP 이후)
- **한국**: 카카오페이 (데어무드 참고)
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

## 섹션 8. 참석의사 (RSVP)

### 8-1. RSVP 항목 설정
- **작동**: 게스트가 제출할 정보 항목을 ON/OFF로 선택
- **항목**: 이름 / 연락처 / 참석 인원 / 식사 여부 / 버스 탑승 여부 / 기타 메시지
- **필수/선택**: 각 항목별 필수 여부 토글
- **MVP**: ✅ 포함 | **우선순위**: P1

### 8-2. RSVP 접속 시 팝업 안내
- **작동**: 게스트가 청첩장 URL에 처음 접속하면 RSVP 참여 유도 팝업 자동 표시
- **옵션**: 팝업 표시 ON/OFF
- **MVP**: ✅ 포함 | **우선순위**: P2

### 8-3. 집계 대시보드
- **작동**: 마이페이지 → 내 청첩장 → RSVP 탭에서 응답 목록 및 집계 확인
- **표시 정보**: 총 참석 예정 인원, 식사 여부 통계, 응답 목록 테이블
- **MVP**: ✅ 포함 | **우선순위**: P2

### 8-4. 엑셀 다운로드
- **작동**: 대시보드에서 "다운로드" 버튼 클릭 시 CSV 또는 XLSX 파일로 응답 데이터 내보내기
- **MVP**: ✅ 포함 | **우선순위**: P2

### 8-5. Zalo 알림
- **작동**: 게스트가 RSVP 제출 시 신랑·신부에게 Zalo 메시지 알림 발송
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

## 섹션 9. 방명록

### 9-1. 방명록 작성
- **작동**: 게스트가 이름(선택)과 축하 메시지를 입력·제출
- **비밀번호 없음**: 별도 인증 없이 누구나 작성 가능
- **MVP**: ✅ 포함 | **우선순위**: P1

### 9-2. 공개·비공개 설정
- **작동**: 공개 시 모든 게스트가 방명록 목록 열람 가능 / 비공개 시 신랑·신부만 마이페이지에서 확인 가능
- **MVP**: ✅ 포함 | **우선순위**: P2

### 9-3. 디자인 타입
- **작동**: 방명록 UI 스타일 선택
- **옵션**:
  - 기본형: 일반 리스트 카드
  - 포스트잇: 노란 포스트잇 스티커 스타일
  - 버튼식: "방명록 보기" 버튼 → 탭 시 팝업으로 열림
- **MVP**: ✅ 기본형 포함 | **우선순위**: P2

### 9-4. 날짜 숨김
- **작동**: 방명록 각 항목에 표시되는 작성 날짜·시간을 숨김 처리
- **MVP**: ✅ 포함 | **우선순위**: P3

## 섹션 10. 추가 콘텐츠 (v2 이후)

### 10-1. 인터뷰
- **작동**: 신랑·신부가 미리 정해진 질문에 답변을 입력 → 청첩장 내 인터뷰 섹션으로 표시
- **샘플 질문**: "처음 만났을 때 기억은?", "결혼을 결심한 이유는?" 등
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 10-2. 타임라인
- **작동**: 커플의 만남부터 결혼까지 날짜별 이야기를 사진+텍스트로 입력 → 세로 타임라인으로 표시
- **항목 추가**: "이야기 추가" 버튼으로 무제한 항목 추가
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 10-3. 배경음악
- **작동**: 제공된 웨딩 음악 목록에서 선택 또는 YouTube URL로 직접 추가
- **옵션**: Romantic memories / Love story / Sweet wedding melody 등 7종 기본 제공
- **자동재생**: 게스트 접속 시 자동재생 여부 설정 (브라우저 정책으로 음소거 후 자동재생)
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 10-4. 유튜브 동영상 삽입
- **작동**: YouTube 영상 URL 입력 → 청첩장 내 지정 위치에 임베드 플레이어 표시
- **비율**: 기본(16:9) / 세로(9:16) / 정방형(1:1) 선택
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 10-5. 중간 사진
- **작동**: 청첩장 섹션과 섹션 사이에 전면 이미지를 삽입하는 구분용 사진
- **최대**: 5장
- **순서**: 순서 변경 탭에서 원하는 위치로 이동
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 10-6. 오프닝 애니메이션
- **작동**: 게스트가 청첩장 URL 접속 시 전면 인트로 애니메이션이 3~5초 재생된 후 본 청첩장으로 전환
- **배경 톤**: 반투명 / 불투명 선택
- **멘트**: 기본 문구 또는 직접 입력
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

### 10-7. 하객 사진 QR
- **작동**: 예식장에서 QR 코드 스캔 → 하객이 직접 찍은 사진을 업로드 → 신랑·신부 구글 드라이브로 자동 수집
- **연동**: Google Drive OAuth
- **MVP**: ❌ 제외(v2) | **우선순위**: P3

## 섹션 11. 공유 설정

### 11-1. 카카오(Zalo) 공유 썸네일
- **작동**: Zalo/Facebook 메신저로 공유 시 미리보기 카드에 표시될 사진·제목·설명 설정
- **기본값**: 커버 사진 + "○○ & ○○ 결혼식에 초대합니다" 자동 생성
- **직접 설정**: 사진 업로드, 제목·설명 직접 입력 가능
- **MVP**: ✅ 포함 | **우선순위**: P1

### 11-2. URL 공유 썸네일
- **작동**: 일반 링크(SMS, 이메일 등) 공유 시 미리보기 카드 설정 (Open Graph 메타태그)
- **비율 제한**: 가로 2:1 비율 이미지만 적용
- **캐시 지연**: 변경 후 30~180분 후 반영 (SNS 캐시 정책)
- **MVP**: ✅ 포함 | **우선순위**: P1

### 11-3. 공유 버튼 표시
- **작동**: 게스트 청첩장 화면 하단 고정 바에 공유 버튼 표시
- **옵션**: Zalo / Facebook / 링크 복사
- **MVP**: ✅ 포함 | **우선순위**: P1

## MVP 우선순위 요약

| 우선순위 | 기준 | 섹션 |
|---|---|---|
| **P1** (출시 필수) | 없으면 서비스 불가 | 기본정보, 예식일시, 예식장소, 갤러리, 공유 |

| **P3** (v2) | 차별화 기능 | 이펙트, 배경음악, 인터뷰, 타임라인, 간편송금 |
