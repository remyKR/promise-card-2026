# 마이페이지 — 화면 기능 명세서

> 경로: `/my` · `/my/cards` · `/my/account`  
> 인증: 로그인 필요 (비로그인 접근 시 `/login`으로 리디렉트)  
> 버전: v1.0 | 작성일: 2026-04-23

---

## 1. 마이페이지 허브 (`/my`)

### 화면 구조

```
┌─────────────────────────────────┐
│  [네비게이션 바]                  │
├─────────────────────────────────┤
│  [프로필 영역]                   │
│  소셜 프로필 사진 + 이름 + 전화번호│
├─────────────────────────────────┤
│  [메뉴 목록]                     │
│  💌 내 청첩장        [>]         │
│  ⚙️  계정 정보 수정   [>]         │
│  ❓  고객센터         [>]         │
│  🚪 로그아웃                     │
└─────────────────────────────────┘
```

### 기능 상세

#### 1-1. 프로필 영역
- **표시**: 소셜 로그인 프로필 사진, 소셜 이름, 인증된 전화번호
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 1-2. 내 청첩장 메뉴
- **작동**: 탭 → `/my/cards`로 이동
- **뱃지**: 청첩장 수 표시 (예: "2개")
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 1-3. 계정 정보 수정 메뉴
- **작동**: 탭 → `/my/account`로 이동
- **MVP**: ✅ 포함 | **우선순위**: P2

#### 1-4. 고객센터 메뉴
- **작동**: 탭 → `/support`로 이동 (외부 링크 또는 내부 페이지)
- **MVP**: ✅ 포함 | **우선순위**: P2

#### 1-5. 로그아웃
- **작동**: 탭 → "로그아웃하시겠습니까?" 확인 모달 → 확인 시 Supabase 세션 삭제 → `/`으로 이동
- **MVP**: ✅ 포함 | **우선순위**: P1

## 2. 내 청첩장 목록 (`/my/cards`)

### 화면 구조

```
┌─────────────────────────────────┐
│  [← 뒤로] 내 청첩장              │
├─────────────────────────────────┤
│  [+ 새 청첩장 만들기] ← 버튼    │
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │ 커버 이미지 썸네일       │    │
│  │ 신랑 ♥ 신부              │    │
│  │ 2026.05.01 | 임시저장   │    │
│  │ [편집] [공유] [삭제]    │    │
│  └─────────────────────────┘    │
│                                  │
│  ┌─────────────────────────┐    │
│  │ ...                     │    │
│  └─────────────────────────┘    │
│                                  │
│  (청첩장 없을 때)                │
│  "아직 만든 청첩장이 없어요"     │
│  [지금 만들기] 버튼              │
└─────────────────────────────────┘
```

### 기능 상세

#### 2-1. 새 청첩장 만들기 버튼
- **작동**: 탭 → `/create`로 이동
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 2-2. 청첩장 카드

| 요소 | 내용 |
|---|---|
| 썸네일 | 커버 이미지 (없으면 테마 기본 이미지) |
| 이름 | 신랑이름 ♥ 신부이름 |
| 날짜 | 예식일 (없으면 "날짜 미설정") |
| 상태 뱃지 | 임시저장 (draft) / 공개중 (published) |

- **MVP**: ✅ 포함 | **우선순위**: P1

#### 2-3. 편집 버튼
- **작동**: 탭 → `/create?id=[card_id]`로 이동 (해당 청첩장 에디터로)
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 2-4. 공유 버튼
- **조건**: published 상태일 때만 활성화
- **작동**: 탭 → 공유 URL 클립보드 복사 + 공유 옵션 시트 표시
  - 링크 복사
  - Facebook 공유
  - Zalo 공유
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 2-5. 삭제 버튼
- **작동**: 탭 → "청첩장을 삭제하면 복구할 수 없습니다. 삭제하시겠습니까?" 확인 모달
- **확인 시**: DB에서 삭제 (연결된 갤러리·RSVP·방명록 모두 삭제됨)
- **삭제 완료**: "청첩장이 삭제되었습니다" 토스트, 목록에서 제거
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 2-6. 빈 상태 (청첩장 0개)
- **표시**: 일러스트 + "아직 만든 청첩장이 없어요" + "지금 만들기" 버튼
- **MVP**: ✅ 포함 | **우선순위**: P1

## 3. 계정 정보 수정 (`/my/account`)

### 화면 구조

```
┌─────────────────────────────────┐
│  [← 뒤로] 계정 정보              │
├─────────────────────────────────┤
│  소셜 계정 연동 현황             │
│  ✅ Google (이메일)             │
│  또는 ✅ Facebook (이름)        │
├─────────────────────────────────┤
│  전화번호 [010-XXXX-XXXX] [변경] │
├─────────────────────────────────┤
│  마케팅 수신 동의  [ON/OFF]      │
├─────────────────────────────────┤
│  ─────────────────────          │
│  [회원 탈퇴]  ← 텍스트 버튼     │
└─────────────────────────────────┘
```

### 기능 상세

#### 3-1. 소셜 계정 연동 현황
- **표시**: 로그인에 사용한 소셜 계정 이메일 또는 이름 표시
- **수정 불가**: 소셜 계정은 서비스에서 변경 불가 (소셜 계정 설정에서 변경)
- **MVP**: ✅ 포함 | **우선순위**: P2

#### 3-2. 전화번호 변경
- **작동**: [변경] 탭 → 새 전화번호 입력 → OTP 재인증 → 저장
- **MVP**: ✅ 포함 | **우선순위**: P2

#### 3-3. 마케팅 수신 동의
- **작동**: 토글 ON/OFF, 즉시 저장
- **기본값**: OFF
- **MVP**: ✅ 포함 | **우선순위**: P2

#### 3-4. 회원 탈퇴
- **작동**: 탭 → "탈퇴하면 모든 청첩장 데이터가 삭제됩니다. 정말 탈퇴하시겠습니까?" 확인 모달
- **확인 시**: Supabase 계정 삭제, 모든 데이터 삭제, `/`로 이동
- **주의**: 삭제 전 "청첩장이 N개 있습니다" 경고 표시
- **MVP**: ✅ 포함 | **우선순위**: P2

## 공통: 에러 상태

| 상황 | 처리 |
|---|---|
| 비로그인 접근 | `/login?redirect=/my`로 리디렉트 |
| 네트워크 오류 | "불러오는 중 오류가 발생했습니다. 새로고침 해주세요." |
| 삭제 실패 | "삭제에 실패했습니다. 다시 시도해주세요." 토스트 |
