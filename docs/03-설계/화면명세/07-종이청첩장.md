# 종이 청첩장 — 화면 기능 명세서

> 경로: `/create/[cardId]/paper` (에디터에서 진입)
> 인증: 로그인 + 청첩장 소유자 필요
> 버전: v1.0 | 작성일: 2026-05-04
> MVP 단계: ❌ (v2~v3 도입)

---

## 도입 배경

### 베트남 결혼 시장 인사이트
- 어르신·친척에게 **종이 청첩장 전달 문화** 여전히 강함
- 도시 커플 **40~60%**가 사진 포함 종이 청첩장 사용
- 베트남 5개 경쟁사(CineLove·Chung Đôi·Vesey·meWedding·iWedding) 중 **모바일+종이 통합 사례 없음** → 블루오션

### 사업적 가치
- **매출 7~10배 증가**: 디지털 200K~500K VND vs 종이+디지털 패키지 1.5M~3.5M VND
- **모방 방어막**: 인쇄소 파트너십·물류 노하우는 6~12개월 진입 장벽
- **사용자 lock-in**: 종이 인쇄·배송 후엔 카피본 이전 의미 X

| Phase | 시점 | 범위 |

## 섹션별 명세

### 1. 사이즈 선택 (Chọn kích thước)

| 사이즈 | 규격 | 분위기 | 가격 등급 |
|---|---|---|---|
| **A6** | 105×148mm | 작고 흔함 | 저가 |
| **A5** | 148×210mm | 사진 포함, 중간 | 표준 |
| **바이폴드** | 210×210mm 펼침 (105×210mm 접힘) | 프리미엄 | 중상 |
| **트라이폴드** | 315×210mm 펼침 (105×210mm 접힘 3폭) | 일정·지도 포함 | 상 |

- 미리보기 썸네일 비교
- 가격 차이 명시
- **MVP 진입**: A5 + 바이폴드 (2종)
- **확장**: A6, 트라이폴드 (Phase 3 후)

### 1-A. 페이지 수별 유형 (3종) — 기능 고정

> 종이 청첩장은 페이지 수(면 수)에 따라 포함 가능한 기능이 다름.
> 사용자가 페이지 수 유형을 먼저 선택 → 해당 유형의 고정 기능 세트만 표시.

#### 유형 ① **2페이지** (1장 양면, A6 또는 A5 싱글 카드) — 미니멀

```
┌─────────┐    ┌─────────┐
│  면 1   │    │  면 2   │
│ (앞면)  │    │ (뒷면)  │
└─────────┘    └─────────┘
```

| 면 | 고정 기능 |
|---|---|
| **면 1 (앞면)** | 신랑·신부 이름 ⭐<br>메인 커버 사진 ⭐<br>예식 날짜 ⭐ |
| **면 2 (뒷면)** | 예식 일시 한 줄<br>예식 장소 한 줄<br>**메인 청첩장 QR** ⭐<br>**VietQR (송금)** (선택) |

- **포함 가능**: 이름·날짜·장소·QR 2종 (메인·VietQR)
- **불가**: 인사말 본문·갤러리·양가 부모·약도·기타 QR
- **가격 등급**: 저가 (가까운 친척만 초대용)
- **추천 사이즈**: A6 (105×148mm) 또는 A5 (148×210mm)

#### 유형 ② **4페이지** (2장 양면 또는 바이폴드) — 표준 ⭐

```
┌────┐  ┌────┐         ┌─────────┐
│면1 │  │면2 │   또는   │ 1│2│3│4 │ (바이폴드 펼침)
│면3 │  │면4 │         └─────────┘
└────┘  └────┘
```

| 면 | 고정 기능 |
|---|---|
| **면 1 (표지 앞)** | 신랑·신부 이름 ⭐<br>메인 커버 사진 ⭐<br>예식 날짜 ⭐ |
| **면 2 (표지 뒤 또는 속지 1)** | 인사말 (Lời mời) — 요약 또는 풀버전<br>(미입력 시 영역 자동 숨김) |
| **면 3 (속지 2)** | 예식 일시 상세<br>예식 장소 상세 (예식장명·홀·전화)<br>간이 지도 이미지 + **지도 QR**<br>양가 부모 안내 (선택) |
| **면 4 (뒷표지)** | 감사 메시지<br>**메인 청첩장 QR** ⭐<br>**VietQR (송금)** (선택)<br>**RSVP QR** (선택) |

- **포함 가능**: 면 1~4의 모든 고정 기능
- **선택 가능 (속지 또는 뒷면)**: 지도·양가 부모·VietQR·RSVP QR·방명록 QR
- **불가**: 갤러리 사진 다중·음악 QR (공간 부족)
- **가격 등급**: 표준 ⭐ (가장 흔함)
- **추천 사이즈**: A5 (148×210mm) 2장 또는 바이폴드 (210×210mm)

#### 유형 ③ **6+페이지** (3장+ 또는 트라이폴드·소책자) — 프리미엄

```
┌────┐ ┌────┐ ┌────┐         ┌────────────┐
│ 1  │ │ 3  │ │ 5  │   또는   │ 1│2│3│4│5│6 │ (트라이폴드)
│ 2  │ │ 4  │ │ 6  │         └────────────┘
└────┘ └────┘ └────┘
```

| 면 | 고정 기능 |
|---|---|
| **표지 (면 1)** | 이름 ⭐ + 메인 사진 ⭐ + 예식 날짜 ⭐ |
| **속지 1 (면 2~3)** | 인사말 풀버전 (Lời mời)<br>예식 일시·장소 상세<br>양가 부모 안내 |
| **속지 2 (면 4~5)** | 갤러리 사진 1~3장 + **갤러리 QR**<br>지도 + 약도 사진 + **지도 QR**<br>축의금 안내 + 계좌 정보 + **VietQR** |
| **뒷표지 (면 6)** | 감사 메시지<br>**메인 청첩장 QR** ⭐<br>**RSVP QR**<br>**방명록 QR**<br>**음악 QR** (선택) |

- **포함 가능**: 모든 고정 기능 + QR 7종
- **가격 등급**: 프리미엄
- **추천 사이즈**: A5 3장 / 트라이폴드 (315×210mm) / 소책자 (4~8장)

### 2. 자동 사이즈·디자인 변환

#### 2-1. 콘텐츠 매핑
- 모바일 청첩장 무한 세로 스크롤 → 종이 한정 면적
- 자동 콘텐츠 추출 + 사용자 선택
- 모든 섹션 변환 X — 표지·속지에 적합한 항목만

#### 2-2. 디자인 변환
- **색상**: RGB → CMYK 자동 변환 (Sharp + ICC 프로파일)
- **폰트**: 인쇄용 서브셋 (베트남어 디아크리틱 지원 확인)
- **여백**: 자동 5~10mm
- **재단선 (Bleed)**: 3mm
- **인쇄 마크**: 재단선·등록 마크 자동 추가

#### 2-3. 디자이너 검수 (Phase 4)
- 자동 변환 후 디자이너 1차 검수 (옵션, 추가 비용)
- 자연스러운 종이 디자인 보정

### 3. 모바일 청첩장 기능 → 종이 청첩장 매핑

> 사용자가 모바일 청첩장에서 입력한 데이터·기능을 종이 청첩장에 자동 변환·반영.
> 12개 모바일 섹션 중 종이 적용 가능한 기능을 3가지 카테고리로 분류.

#### 3-1. 직접 변환 (모바일 데이터 → 종이 자동 채움)

| 모바일 섹션 | 종이 적용 위치 | 비고 |
|---|---|---|
| 신랑·신부 이름 (섹션 2) | 표지 + 속지 | 자동 |
| 부모님 이름 (섹션 2) | 속지 양가 안내 | 미입력 시 자동 숨김 |
| 메인 커버 사진 (섹션 1 Ảnh cưới) | 표지 | 자동 |
| 글자 색상 (섹션 1 Màu chữ) | 종이 텍스트 색상 (CMYK 변환) | 3색 매핑 |
| 인사말 제목 (섹션 3-1) | 속지 헤더 | 빈 값 허용 |
| 인사말 본문 (섹션 3-2) | 속지 본문 | 리치 텍스트 → 종이용 변환 |
| 예식 날짜·시간 (섹션 4) | 표지·속지 | 자동 포맷 |
| 예식장 정보 (섹션 5) | 속지 | 예식장명·층·홀·전화번호 |
| 약도 사진 (섹션 5-4) | 속지 | 직접 업로드한 약도 |
| 갤러리 사진 1~3장 (섹션 6) | 속지·뒷면 | 사용자 선택 |
| 축의금 안내 메시지 (섹션 7-2) | 속지 | 리치 텍스트 → 종이용 |
| 축의금 계좌 정보 (섹션 7-3) | 속지 | 선택적 (대신 VietQR로 대체 권장) |

#### 3-2. QR 코드 통합 (모바일 인터랙션 → 종이 진입점) ⭐

종이에 QR 코드 인쇄 → 게스트가 스캔 → 모바일 청첩장의 해당 페이지로 직접 진입.
**종이의 한계(인터랙션 불가)를 모바일로 보완하는 핵심 차별화**.

| QR 종류 | 연결 페이지 | 권장 표시 위치 |
|---|---|---|
| **메인 청첩장 QR** ⭐ (필수) | 모바일 청첩장 메인 URL | 표지 또는 뒷표지 |
| **RSVP QR** | 참석 확인 페이지 (섹션 9) | 속지 하단 |
| **방명록 QR** | Lời chúc 작성 페이지 (섹션 10) | 속지 또는 뒷면 |
| **갤러리 QR** | 모바일 갤러리 전체 (섹션 6) | 속지 (갤러리 사진 옆) |
| **음악 QR** | 배경음악 재생 (섹션 8) | 속지 (장식 모티브 옆) |
| **지도 QR** | Google Maps 길안내 딥링크 | 속지 (지도 이미지 옆) |
| **VietQR (축의금 송금)** ⭐ | VietQR.io 이미지 (섹션 7-4) | 속지 (계좌 카드 내부) |

→ 사용자가 QR 종류별 ON/OFF 토글로 인쇄 여부 선택.

#### 3-3. 표지·속지·뒷면 콘텐츠 항목 선택

##### 표지 (Trang bìa) — 필수
- ✅ 신랑·신부 이름
- ✅ 메인 커버 사진
- ✅ 예식 날짜
- 🔘 부제목·인사말 한 줄
- 🔘 메인 청첩장 QR (선택, 보통 뒷표지에 배치)
- 🔘 장식 모티브 (꽃·잎사귀·기하학)

##### 속지 (Trang trong) — 사용자 선택
- 🔘 인사말 (Lời mời) 풀버전 또는 요약
- 🔘 예식 일시 상세
- 🔘 예식 장소 상세 (예식장명·층·홀·전화번호)
- 🔘 지도 (정적 이미지) + 지도 QR
- 🔘 약도 사진 (사용자 업로드)
- 🔘 양가 부모님 안내
- 🔘 갤러리 사진 1~3장 + 갤러리 QR
- 🔘 축의금 안내 메시지 + VietQR (송금)
- 🔘 RSVP QR (참석 확인)
- 🔘 방명록 QR (Lời chúc 작성)
- 🔘 음악 QR (배경음악 재생)

##### 뒷면 (Trang sau) — 사용자 선택
- 🔘 신랑·신부 사진 1장
- 🔘 감사 메시지
- 🔘 메인 청첩장 QR (대표 위치, 권장 ⭐)

→ 항목별 ON/OFF 체크 → 자동 레이아웃 배치

#### 3-4. 종이 미적용 모바일 기능 (참고)

다음 모바일 기능은 종이의 정적 특성상 변환 불가 → QR 또는 별도 처리:

| 모바일 기능 | 종이 미적용 이유 | 대체 방안 |
|---|---|---|
| 배경 효과 (Hiệu ứng nền 4종 파티클) | 정적 매체, 애니메이션 X | 정적 배경 패턴 (선택) |
| D-Day 카운트다운 | 인쇄 후 실시간 업데이트 X | 표지에 단순 날짜만 |
| 갤러리 타입 (Vuốt/Lưới) | 종이는 그리드 고정 | 그리드 레이아웃만 |
| 마스터 토글·게스트 복사 버튼 | 인터랙션 X | QR로 대체 |
| 공유 카드 (OG 메타태그) | 종이 무관 | 메인 청첩장 QR로 대체 |
| 모바일 순서 변경 모달 | 종이 별도 순서 결정 | 표지·속지·뒷면 순서로 단순화 |
| RSVP 항목 6종 (이름·전화·인원·식사 등) | 종이에 직접 입력 X | RSVP QR로 모바일 폼 진입 |
| 방명록 작성 폼 | 직접 작성 X | 방명록 QR로 모바일 폼 진입 |

### 4. 데이터 동기화
- 모바일 청첩장 입력 데이터 자동 가져오기
- 사용자가 종이용으로 별도 수정 가능 (선택)
- 모바일 변경 시 종이 미반영 (이미 인쇄 후라 별개)

### 5. 미리보기

- **2D 평면 미리보기**: 표지·속지·뒷면 펼친 모습
- **접힘 미리보기** (바이폴드·트라이폴드): 접힌 모습
- **색상 안내**: "모니터 색상과 실제 인쇄 색상은 차이가 있을 수 있습니다" 고지
- **확대·축소** 가능

### 6. 옵션 선택 (Phase 3)

### 7. 결제 (Phase 3, 결제 PG 도입과 동시)

- 베트남 PG 통합: ZaloPay / MoMo / VietQR / 카드
- 결제 시점: 본인쇄 직전 100% 또는 미리 50% + 본인쇄 직전 50%
- **환불 정책**:
  - PDF 변환 후 취소: 30% 위약금
  - 본인쇄 시작 후: 환불 X
  - 인쇄 미스 (인쇄소 책임): 재인쇄 무료

### 8. PDF 변환 (Phase 2~)

#### 8-1. 인쇄용 PDF 사양
- **색상**: CMYK
- **해상도**: 300 DPI
- **재단선 (Bleed)**: 3mm
- **폰트**: 임베드 (subset)
- **인쇄 마크**: 재단선·등록 마크
- **페이지**: 표지·속지·뒷면 분리

#### 8-2. 기술 스택
- **Puppeteer + headless Chrome**: HTML → PDF
- **Sharp**: 이미지·CMYK 변환
- **PDF-lib / PDFKit**: PDF 후처리
- **ICC Profile**: CMYK 색상 프로파일 적용

### 9. 컬러 감리 (Phase 4 프리미엄 옵션)

- 인쇄소 작업자가 첫 인쇄본 색상 정확도 검수
- 추가 비용: +300~500K VND
- 시간: 24~48시간

### 10. 테스트 인쇄 + 고객 컨펌 (Phase 4 프리미엄 옵션)

#### 10-1. 테스트 인쇄
- 본 인쇄 전 1~2장 시험 인쇄
- 추가 비용: +200~500K VND
- 시간: 1~2일

#### 10-2. 테스트본 고객 배송
- 고객이 실물 확인 후 컨펌
- 추가 비용: +50~150K VND
- 시간: +3~7일

#### 10-3. 고객 컨펌
- 만족 → 본인쇄 진행
- 수정 요청 → 디자인 수정 후 재테스트 (추가 비용)

### 11. 본 인쇄

- 인쇄소 본 작업
- 시간: 50장 기준 1~3일
- 품질 검수 후 포장
