# 청첩장 템플릿 설계

> 버전: v1.0 | 최종수정: 2026-04-23  
> MVP 3종 템플릿 정의. DB의 `theme_id` 값과 `settings` jsonb 구조에 직접 매핑됨.

---

## 템플릿 개요

MVP에서 제공하는 3가지 테마. 레이아웃 구조는 동일하고 색상·폰트·장식 요소만 다르다.

| theme_id | 이름 (베트남어) | 분위기 | 타겟 |
|---|---|---|---|
| `minimal` | Tối giản (미니멀) | 깔끔, 모던, 여백 | 도시 커플, 미니멀 선호 |
| `romantic` | Lãng mạn (로맨틱) | 화사, 따뜻, 플로럴 | 여성 취향, 감성적 |
| `classic` | Cổ điển (클래식) | 격식, 전통, 고급감 | 가족 중심, 전통 혼례 |

## settings jsonb 구조

`invitation_cards.settings` 컬럼에 저장되는 사용자 커스터마이징 값.

```typescript
interface CardSettings {
  themeId: 'minimal' | 'romantic' | 'classic'
  // 사용자가 기본값에서 덮어쓴 값만 저장 (미변경 시 undefined)
  bgColor?: string        // 배경색 HEX
  accentColor?: string    // 강조색 HEX
  headingFont?: string    // 제목 폰트 이름
  bodyFont?: string       // 본문 폰트 이름
  showParentsNames: boolean   // 부모님 이름 표시 여부
  brideFirst: boolean         // 신부측 먼저 표시 여부
  showCalendar: boolean       // 달력 표시 여부 (P2)
  showCountdown: boolean      // D-Day 카운트다운 여부 (P2)
  galleryStyle: 'swipe'       // MVP는 swipe 고정, v2에서 'grid' 추가
  showRsvp: boolean           // RSVP 섹션 표시 여부
  showGuestbook: boolean      // 방명록 섹션 표시 여부
  showBankAccounts: boolean   // 계좌번호 섹션 표시 여부
}
```

## 폰트 로딩 전략

베트남어 텍스트가 포함되므로 Vietnamese subset 지원 필수.

| 폰트 | 출처 | Vietnamese 지원 |
|---|---|---|
| Cormorant Garamond | Google Fonts | ✅ |
| Playfair Display | Google Fonts | ✅ |
| Libre Baskerville | Google Fonts | ✅ |
| Be Vietnam Pro | Google Fonts | ✅ (베트남 전용 폰트) |

```typescript
// next/font/google 사용 (자동 최적화)
import { Cormorant_Garamond, Be_Vietnam_Pro } from 'next/font/google'
```

## Figma 작업 순서 (디자이너용)

1. `minimal` 템플릿을 먼저 완성 (기준 템플릿)
2. `romantic`, `classic`은 minimal 구조를 복사 후 색상·폰트·장식만 변경
3. 각 템플릿 커버 화면을 랜딩 페이지 미리보기에 사용할 썸네일로도 내보내기
