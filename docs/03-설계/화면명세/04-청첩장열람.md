# 청첩장 열람 — 화면 기능 명세서

> 경로: `/card/[code]`  
> 인증: 불필요 (공개 URL, 누구나 접근)  
> 대상: 청첩장 받은 하객  
> 버전: v1.0 | 작성일: 2026-04-23

---

## 화면 구조

```
┌─────────────────────────────────┐
│  [커버 섹션]                     │  ← 신랑·신부 이름, 예식일, 커버 이미지
├─────────────────────────────────┤
│  [인사말 섹션]                   │  ← 인사 문구
├─────────────────────────────────┤
│  [예식 일시 섹션]                 │  ← 날짜·시간·달력·D-Day
├─────────────────────────────────┤
│  [예식 장소 섹션]                 │  ← 주소·지도·내비게이션
├─────────────────────────────────┤
│  [갤러리 섹션]                   │  ← 커플 사진
├─────────────────────────────────┤
│  [계좌번호 섹션]                  │  ← 축의금 안내
├─────────────────────────────────┤
│  [RSVP 섹션]                     │  ← 참석 여부 응답
├─────────────────────────────────┤
│  [방명록 섹션]                   │  ← 축하 메시지
├─────────────────────────────────┤
│  [공유 버튼 (하단 고정)]          │
└─────────────────────────────────┘
```

- 섹션 순서는 에디터에서 설정한 순서대로 렌더링
- 활성화된 섹션만 표시 (비활성화된 섹션은 숨김)
- 모바일 우선 단일 컬럼 레이아웃

---

## 접근 제어

| 상태 | 동작 |
|---|---|
| 정상 공개 | 청첩장 전체 표시 |
| 잠금 (`locked_at` 경과) | "청첩장이 종료되었습니다" 안내 페이지 표시 |
| 미발행 (`status = draft`) | "존재하지 않는 청첩장입니다" 404 페이지 표시 |
| 잘못된 코드 | 404 페이지 표시 |

---

## 1. 커버 섹션

- **표시 내용**: 신랑·신부 이름 (한자·영문 설정 따름), 예식일, 커버 이미지(또는 배경색)
- **테마 적용**: 에디터에서 선택한 글꼴·배경색·강조색 전체 적용
- **MVP**: ✅ 포함 | **우선순위**: P1

---

## 2. 인사말 섹션

- **표시 내용**: 에디터에서 입력한 인사말 텍스트 (줄바꿈 유지)
- **스타일**: 중앙 정렬, 선택한 글꼴 적용
- **MVP**: ✅ 포함 | **우선순위**: P1

---

## 3. 예식 일시 섹션

### 3-1. 날짜·시간 표시
- **표시**: 2026년 5월 1일 금요일 오후 1시 30분
- **MVP**: ✅ 포함 | **우선순위**: P1

### 3-2. 달력 표시
- **작동**: 해당 월 달력을 표시하고 예식 날짜를 강조 표시
- **MVP**: ✅ 포함 | **우선순위**: P2

### 3-3. D-Day 카운트다운
- **작동**: "결혼식까지 D-23" 형식으로 실시간 카운트다운
- **예식 당일**: "오늘이 결혼식 날입니다 🎉"
- **예식 후**: D+N 표시 (또는 숨김)
- **MVP**: ✅ 포함 | **우선순위**: P2

## 4. 예식 장소 섹션

### 4-1. 장소 정보 표시
- **표시**: 예식장명, 홀 이름, 주소, 전화번호(클릭 시 전화 연결)
- **MVP**: ✅ 포함 | **우선순위**: P1

### 4-2. 지도 표시
- **구현**: Google Maps Embed API (iframe)
- **표시**: 예식장 위치를 마커로 표시
- **MVP**: ✅ 포함 | **우선순위**: P1

### 4-3. 내비게이션 바로가기 버튼
- **버튼 목록**:
  - Google Maps → `https://maps.google.com/?q=위도,경도`
  - Naver Maps → `nmap://route/car?dlat=...` (딥링크)
  - KakaoMap → `kakaomap://route?ep=위도,경도` (딥링크)
- **모바일 환경**: 앱이 없으면 웹 버전으로 폴백
- **MVP**: ✅ 포함 | **우선순위**: P1

### 4-4. 교통 수단 안내
- **표시**: 에디터에서 입력한 대중교통·버스·지하철 안내 텍스트
- **MVP**: ✅ 포함 | **우선순위**: P2

- **닫기**: X 버튼 또는 팝업 바깥 영역 탭
- **MVP**: ✅ 포함 | **우선순위**: P2

## 6. 축의금 섹션 (Mừng cưới)

- **연동**: 에디터 섹션 7의 데이터 직접 표시 (자세한 사양은 [02-에디터.md](02-에디터.md) 섹션 7 참조)
- **마스터 토글 OFF 시**: 섹션 자체 미노출 (DOM에서 제거)
- **표시 구조**: 안내 메시지 + 계좌 카드 리스트 (세로 스택, 작성자가 ▲▼로 정한 순서대로)
- **계좌 카드 내용**:
  - 계좌 이름 (있으면 표시, 없으면 생략)
  - 은행 로고 + 은행명
  - 예금주명
  - 계좌번호 (4자리 단위 공백 표시 — 예: `0011 0023 4567`)
  - **[복사] 버튼**: 클립보드 복사 → "Đã sao chép" 토스트
  - **VietQR 코드** (작성자가 토글 ON 시): 자동 생성, 탭 시 풀스크린 모달로 확대
- **MVP**: ✅ 포함 | **우선순위**: P1

## 7. RSVP 섹션

> RSVP = 참석 여부를 미리 알려달라는 요청

### 화면 구조

```
"참석 여부를 알려주세요"

이름 [              ]
연락처 [            ]
참석 여부  ○ 참석   ○ 불참
참석 인원 [숫자 선택]
식사 여부  ○ 예   ○ 아니오
전달사항 [              ]

[제출하기] 버튼
```

### 기능 상세

#### 7-1. 이름 입력
- **필수**: 빈 값 제출 불가
- **최대 길이**: 50자
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 7-2. 연락처 입력
- **형식**: 숫자 입력, 선택 항목
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 7-3. 참석 여부
- **옵션**: 참석 / 불참 (라디오 버튼)
- **필수**: 선택 전 제출 불가
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 7-4. 참석 인원
- **작동**: 참석 선택 시에만 표시, - / + 버튼으로 조절, 최소 1명
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 7-5. 식사 여부
- **옵션**: 예 / 아니오
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 7-6. 전달사항
- **형식**: 텍스트 영역, 선택 항목
- **최대 길이**: 300자
- **MVP**: ✅ 포함 | **우선순위**: P1

#### 7-7. 제출
- **성공 시**: "RSVP가 전달되었습니다. 감사합니다!" 메시지 표시, 폼 숨김
- **실패 시**: "제출에 실패했습니다. 다시 시도해주세요." 토스트
- **중복 제출**: 동일 브라우저에서 제출 후 재접속 시 이미 제출했다는 안내 (로컬스토리지 기반)
- **MVP**: ✅ 포함 | **우선순위**: P1

## 8. 방명록 섹션

### 8-1. 방명록 목록
- **표시**: is_public = true인 항목만 표시 (최신순)
- **각 항목**: 작성자 이름, 메시지, 작성일
- **최대 표시**: 최근 20개, 더보기 버튼으로 추가 로드
- **MVP**: ✅ 포함 | **우선순위**: P1

### 8-2. 방명록 작성
```
이름 [              ]
메시지 [            ]
공개 여부  ☑ 공개
[등록하기] 버튼
```

- **이름**: 필수, 최대 50자
- **메시지**: 필수, 최대 200자
- **공개 여부**: 기본값 공개, 체크 해제 시 작성자·청첩장 주인만 열람
- **성공 시**: 목록 상단에 즉시 추가
- **MVP**: ✅ 포함 | **우선순위**: P1
