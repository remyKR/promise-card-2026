## 화면 구조

```
┌─────────────────────────────────────────┐
│  [네비게이션 바]                          │
├─────────────────────────────────────────┤
│  [히어로 섹션]                            │
│  타이틀 + 부제목 + CTA 버튼              │
├─────────────────────────────────────────┤
│  [가치 제안 섹션]                         │
│  아이콘 + 설명 카드 4개                  │
├─────────────────────────────────────────┤
│  [템플릿 미리보기 섹션]                   │
│  청첩장 디자인 3종 슬라이드               │
├─────────────────────────────────────────┤
│  [이용 방법 섹션]                         │
│  3단계 설명                               │
├─────────────────────────────────────────┤
│  [CTA 섹션]                               │
│  "지금 바로 만들기" 버튼                  │
├─────────────────────────────────────────┤
│  [푸터]                                   │
└─────────────────────────────────────────┘
```

---

## 1. 네비게이션 바

- **작동**: 상단 고정(sticky), 스크롤 시 배경 반투명 → 불투명 전환
- **비로그인**: [로고] [청첩장 만들기] [로그인]
- **로그인**: [로고] [청첩장 만들기] [마이페이지]
- **모바일**: 로고 + 햄버거 메뉴(우측)
- **MVP**: ✅ 포함 | **우선순위**: P1

---

## 2. 히어로 섹션

- **타이틀**: "5분 만에 완성하는 모바일 청첩장" (베트남어)
- **부제목**: "Zalo·Facebook으로 바로 공유, 무료"
- **배경**: 청첩장 미리보기 이미지 또는 그라디언트
- **CTA 버튼 (주)**: "청첩장 만들기" → 로그인 상태면 `/create`, 비로그인이면 `/login`으로 이동
- **CTA 버튼 (부)**: "샘플 보기" → 페이지 내 템플릿 미리보기 섹션으로 스크롤
- **MVP**: ✅ 포함 | **우선순위**: P1

---

## 3. 가치 제안 섹션

아이콘 + 제목 + 설명으로 구성된 카드 4개 가로 배열 (모바일은 2×2 그리드)

| # | 아이콘 | 제목 | 설명 |
|---|---|---|---|
| 1 | ⏱ | 5분 완성 | 복잡한 디자인 도구 없이 간단하게 |
| 2 | 📱 | 바로 공유 | Zalo·Facebook으로 링크 한 번에 |
| 3 | 🗺 | 베트남 현지화 | 구글맵 연동, 베트남어 UI |
| 4 | 🎁 | 완전 무료 | 종이 청첩장 비용 0원 |

- **MVP**: ✅ 포함 | **우선순위**: P1

---

## 4. 템플릿 미리보기 섹션

- **제목**: "마음에 드는 디자인을 골라보세요"
- **표시**: 청첩장 템플릿 3종을 모바일 프레임(폰 목업) 안에 표시
- **인터랙션**: 좌우 스와이프 또는 탭으로 템플릿 전환
- **각 템플릿 카드**:
  - 디자인 이름 (예: Classic · Modern · Minimal)
  - "이 디자인으로 만들기" 버튼 → `/login` 또는 `/create?theme=classic`
- **MVP**: ✅ 포함 | **우선순위**: P1

---

## 5. 이용 방법 섹션

- **제목**: "이렇게 만드세요"
- **3단계 표시** (아이콘 + 번호 + 설명):

| 단계 | 설명 |
|---|---|
| 1. 정보 입력 | 신랑·신부 이름, 예식 일시·장소 입력 |
| 2. 사진·디자인 | 커플 사진 올리고 글꼴·색상 선택 |
| 3. 링크 공유 | Zalo·Facebook으로 바로 전송 |

- **MVP**: ✅ 포함 | **우선순위**: P1
