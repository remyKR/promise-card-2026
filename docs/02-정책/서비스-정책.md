# 서비스 정책

> 버전: v1.0 | 최종수정: 2026-04-23

---

## 1. 청첩장 유효 기간

| 구분 | 정책 |
|---|---|
| MVP (무료) | 예식일로부터 **30일 후** 자동 잠금 |
| 잠금 이후 | 열람 불가, 데이터 보존 (삭제 아님) |
| 데이터 보존 | 잠금 후 **90일** 보존 후 자동 삭제 |
| v2 유료 플랜 | 무제한 보관 (법인 설립 후 도입) |

---

## 2. 청첩장 잠금 규칙

| 항목 | 내용 |
|---|---|
| 잠금 기준 | `invitation_cards.locked_at` 컬럼 기준 |
| 자동 잠금 | 예식일 + 30일 → `locked_at` 자동 설정 |
| 수동 잠금 | 에디터에서 사용자가 직접 설정 가능 |
| 잠금 해제 | MVP에서는 불가 (v2 유료 플랜에서 해제 가능) |
| 잠금 전 알림 | 잠금 7일 전 이메일 알림 (v2에서 구현) |

---

## 3. 업로드 제한

| 항목 | MVP 제한 | v2 제한 |
|---|---|---|
| 갤러리 사진 수 | 최대 20장 | 최대 40장 |
| 사진 파일 형식 | JPG, PNG, WebP | 동일 |
| 사진 최대 용량 | 장당 10MB | 동일 |
| 청첩장 수 | 제한 없음 | 동일 |

---

## 4. 금지 콘텐츠

다음 콘텐츠는 업로드 및 입력이 금지됩니다:

- 타인의 초상권·저작권을 침해하는 사진
- 혐오·폭력·음란 콘텐츠
- 허위 정보를 통한 사기 목적 사용
- 스팸·광고성 청첩장 대량 생성

위반 시 사전 경고 없이 계정 정지 및 데이터 삭제 가능.

---

## 5. 공유 URL 규칙

| 항목 | 내용 |
|---|---|
| 공유 코드 형식 | 영문·숫자 8자리 (랜덤 생성) |
| URL 형식 | `promisecard.vn/card/[code]` |
| 공개 여부 | 기본 공개 (URL 아는 사람 누구나 열람 가능) |
| 청첩장 잠금 | 잠금 시 URL 접근 불가 |

---

## 6. RSVP 정책

| 항목 | 내용 |
|---|---|
| 제출 가능 시점 | 청첩장 공개 후 ~ 예식 당일까지 |
| 수정 가능 여부 | 제출 후 수정 불가 (재제출만 가능) |
| 데이터 열람 | 청첩장 소유자만 열람 가능 |
| 데이터 보존 | 청첩장 삭제 시 함께 삭제 |

---

## 7. 방명록 정책

| 항목 | 내용 |
|---|---|
| 작성 가능 | 청첩장 공개 중 누구나 |
| 공개·비공개 | 작성자가 선택, 기본값 공개 |
| 삭제 권한 | 청첩장 소유자만 삭제 가능 |
| 신고 기능 | v2에서 구현 |
