## 1. 요금 정책

| 구분 | MVP | v2 (법인 설립 후) |
|---|---|---|
| 기본 서비스 | **무료** | 무료 (제한적) |
| 유료 플랜 | 없음 | 월정액 또는 건당 결제 |

> MVP 무료 출시 기간 동안의 월 운영 비용. 사전에 인식하고 예산 책정 필요.

| 서비스 | 무료 한도 | 초과 시 비용 | MVP 예상 |
|---|---|---|---|
| **Supabase** | DB 500MB, Storage 1GB, MAU 50,000 | Pro $25/월 | 초기 무료 → 사용자 증가 시 $25 |
| **Vercel** | 취미 플랜 무료 | Pro $20/월 | 초기 무료 |

| **Google Maps API** | $200/월 크레딧 | 사용량 초과 시 과금 | 크레딧 내 무료 (MVP 규모) |
| **Supabase Storage** | 1GB | $0.021/GB | 사진 10장 × 평균 1MB = 약 10MB/카드 |

**월간 총 예상 비용 (MVP 초기)**
- 사용자 100명 이하: **$5 내외** (Twilio만 발생)
- 사용자 500명: **$30 내외** (Supabase Pro 전환 포함)
- 사용자 1,000명: **$70 내외**

> Supabase Storage 1GB = 청첩장 약 100개 분량. 100개 초과 시 스토리지 비용 발생 시작.

## 8. 종이 청첩장 운영 정책 (Phase 2~4)

> 상세 명세: [03-설계/화면명세/07-종이청첩장.md](../03-설계/화면명세/07-종이청첩장.md)

### 8-1. Phase 2 (출시 6~9개월): Concierge MVP — 수동 운영 ⭐
- **주문량**: 첫 100~500건
- **운영 모델**: 1인 사업자가 매 주문 직접 검수
- **처리 절차**:
  1. 주문 수신 (대시보드 알림)
  2. PDF 검수 (색상·여백·텍스트·사진 해상도)
  3. 필요 시 수동 보정 (Adobe Acrobat Pro / Photoshop)
  4. 인쇄소 이메일·메신저 발주
  5. 인쇄소 색상 감리 요청
  6. 인쇄소 검수 통과 → 본인쇄
  7. 배송사 발송 → 추적 번호 수령
  8. 사용자 Zalo·SMS 알림
- **처리 시간**: 1주문당 30~60분
- **1인 처리 가능**: 월 100~150건 (주 20시간 할애 기준)
- **이유**: 시장 검증 + 인쇄소 신뢰 구축 + 시행착오 학습 (자동화 전 데이터 축적)

### 8-2. Phase 3 (12~18개월): 부분 자동화 — 80% 자동
- **주문량**: 500~2,000건
- **자동화**:
  - 95% PDF 색상 자동 변환 (Sharp + ICC ColorProfile)
  - 인쇄소 발주 API 또는 자동 이메일
  - 배송 추적 자동 알림
- **수동 영역**:
  - 5% 예외 케이스만 사람 검수
  - 프리미엄 옵션 (컬러 감리·테스트 인쇄)
- **처리 시간**: 1~2일 (표준 옵션)

### 8-3. Phase 4 (18~24개월): 풀 자동화 + 프리미엄
- **주문량**: 2,000건+
- **자동화**: 99% 자동 처리
- **프리미엄 옵션**: 컬러 감리·테스트 인쇄·고객 컨펌은 수동 유지
- **AI 보강**: 사진 자동 색상 보정·디자이너 검수 옵션
