# 운영 정책

> 버전: v1.0 | 최종수정: 2026-04-23

---

## 1. 요금 정책

| 구분 | MVP | v2 (법인 설립 후) |
|---|---|---|
| 기본 서비스 | **무료** | 무료 (제한적) |
| 유료 플랜 | 없음 | 월정액 또는 건당 결제 |
| 결제 수단 | 없음 | MoMo, ZaloPay, VietQR |

MVP는 100% 무료로 출시하여 초기 사용자를 확보한 후, 법인 설립 완료 시점에 유료 플랜을 도입한다.

---

## 2. 개인정보 처리 방침

### 수집 항목

| 항목 | 수집 목적 | 보존 기간 |
|---|---|---|
| 이메일 (소셜 제공) | 계정 식별 | 회원 탈퇴 시까지 |
| 전화번호 (+84) | Decree 147 인증 | 회원 탈퇴 시까지 |
| 청첩장 내용 | 서비스 제공 | 청첩장 삭제 또는 만료 후 90일 |
| 갤러리 사진 | 서비스 제공 | 청첩장 삭제 또는 만료 후 90일 |

### 제3자 제공

- Google, Facebook OAuth: 로그인 전용, 콘텐츠 미전달
- Supabase: 데이터 저장 인프라 (AWS ap-southeast-1 리전)
- Vercel: 서비스 호스팅

## 6. 인프라 비용 추산

> MVP 무료 출시 기간 동안의 월 운영 비용. 사전에 인식하고 예산 책정 필요.

| 서비스 | 무료 한도 | 초과 시 비용 | MVP 예상 |
|---|---|---|---|
| **Supabase** | DB 500MB, Storage 1GB, MAU 50,000 | Pro $25/월 | 초기 무료 → 사용자 증가 시 $25 |
| **Vercel** | 취미 플랜 무료 | Pro $20/월 | 초기 무료 |
| **Twilio Verify** | 없음 (종량제) | $0.05/건 | 가입 100명: $5, 1,000명: $50 |
| **Google Maps API** | $200/월 크레딧 | 사용량 초과 시 과금 | 크레딧 내 무료 (MVP 규모) |
| **Supabase Storage** | 1GB | $0.021/GB | 사진 10장 × 평균 1MB = 약 10MB/카드 |

**월간 총 예상 비용 (MVP 초기)**
- 사용자 100명 이하: **$5 내외** (Twilio만 발생)
- 사용자 500명: **$30 내외** (Supabase Pro 전환 포함)
- 사용자 1,000명: **$70 내외**

> Supabase Storage 1GB = 청첩장 약 100개 분량. 100개 초과 시 스토리지 비용 발생 시작.

## 7. 콘텐츠 책임

- 청첩장 내 콘텐츠(사진, 텍스트)에 대한 저작권 및 초상권은 업로드한 사용자가 책임짐
- 서비스는 콘텐츠 저장·전달 역할만 수행
- 타인 신고 시 내용 검토 후 조치 (v2에서 신고 기능 구현 예정)
